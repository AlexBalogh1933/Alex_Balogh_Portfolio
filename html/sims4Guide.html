  For my Web Development class, our main project was to create a website for the class. I wanted to build a website that I may use so I decided to create a Sims 4 Guide. It has multiple pages. Some are complete and some that I am still working on. The ones that I have completed are Skills, Cheats, Challenges, and Easter Eggs. The Careers/Jobs page is half complete. It is still missing a lot of the careers but the template for them is there.
